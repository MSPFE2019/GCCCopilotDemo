Copilot Studio Demo

This is an example of how you can use Copilot Studio with a document containing tables and embed it in a custom website. Click on "Open Copilot Studio Agent" to ask questions about the data stored in the Copilot Studio Agent. Please review the document linked below and find something to ask Copilot.

This Copilot Studio is hosted in a GCC tenant with Generative AI enabled, and the document is uploaded into the Copilot Studio Agent. The website is hosted on GitHub using GitHub Pages.

Please review the 2023 CMS Document. After reviewing the document, feel free to ask Copilot Studio any questions about its content.
